Browser

Get browser's information

Jsonic use Brower to detect the browser.

var _name = Jsonic.Browser.Name; //return chrome|safari|msie|firefox|opera
var _prefix = Jsonic.Browser.Prefix; //return webkit|ms|moz
var _version = Jsonic.Browser.Version; //return a string

attrPrefix

Compatibility

Jsonic.Browser.attrPrefix(window,'AudioContext');

noConflict

The same as the noConflict functinon in Jquery.

Jsonic.noConflict();
