What's Jsonic

Jsonic is a small and feature-rich javascript library. It helps to improve the intaction of sound in HTML5 website or apps. With Jsonic, you can compose music, transfer data with urltrasonic channel, make sound wave visualible, do some speech recognition and so on.

Get start

Use Jsonic

Link Jsonic as usual. Jsonic only works on HTML5 browser now.

<script type="text/javascript" src="Jsonic.js"></script>
